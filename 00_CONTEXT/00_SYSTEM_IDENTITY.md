# System Identity

## Project Name
Career Intelligence Operating System

## Purpose
An AI-assisted operating system for executive career management.

## Mission
Centralize every employment opportunity into one trusted system.

## Core Principles
- Human in the loop
- Local First
- SQLite
- Repository Pattern
- Modular Architecture
- Single Source of Truth
- AI Assisted
- No autonomous decisions
- Incremental development

## What this system IS
Executive career operating system.

## What this system IS NOT
- Job scraper.
- CRM.
- Applicant Tracking System.
- Autonomous AI.
